# Thunderbird Viewer

![NET](https://img.shields.io/badge/NET-10.0-green.svg)
![License](https://img.shields.io/badge/License-MIT-blue.svg)
![VS2022](https://img.shields.io/badge/Visual%20Studio-2026-white.svg)
![Version](https://img.shields.io/badge/Version-1.0.2026.0-yellow.svg)]

Mit der Anwendung kannst du Thunderbird E-Mail-Dateien (MBOX-Format) anzeigen und durchsuchen. Die Anwendung bietet eine benutzerfreundliche Oberfläche, um E-Mails zu lesen, Anhänge zu verwalten und nach bestimmten Nachrichten zu suchen.
